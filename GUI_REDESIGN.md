# PSiteDL GUI 重新设计 - 苹果风格

## 设计概述

本次重新设计参考苹果官网设计语言，采用现代简约风格，打造优雅、直观的用户体验。

## 设计原则

### 1. 简洁至上
- 减少视觉噪音，突出核心功能
- 清晰的层次结构
- 留白创造呼吸感

### 2. 直观易用
- 符合用户心智模型
- 明确的操作反馈
- 渐进式信息展示

### 3. 精致细节
- 微妙的动画过渡
- 细腻的阴影和渐变
- 一致的视觉语言

## 界面元素规范

### 圆角设计
- 大卡片：`12px`
- 按钮：`10px`
- 输入框：`10px`
- 小元素：`8px`

### 阴影效果
```css
/* 轻微阴影 - 卡片 */
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04),
            0 1px 2px rgba(0, 0, 0, 0.06);

/* 中等阴影 - 悬浮 */
box-shadow: 0 4px 16px rgba(0, 0, 0, 0.08),
            0 2px 4px rgba(0, 0, 0, 0.06);

/* 强烈阴影 - 弹窗 */
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.12),
            0 4px 8px rgba(0, 0, 0, 0.08);
```

### 毛玻璃效果
```css
backdrop-filter: saturate(180%) blur(20px);
background-color: rgba(255, 255, 255, 0.72);
```

### 字体规范
- **主标题**: SF Pro Display, 28px, Bold (700)
- **副标题**: SF Pro Text, 17px, Regular (400)
- **正文**: SF Pro Text, 15px, Regular (400)
- **按钮**: SF Pro Text, 15px, Medium (500)
- **说明文字**: SF Pro Text, 13px, Regular (400)

> 注：Windows/Linux 使用系统默认无衬线字体替代

## 交互设计

### 按钮状态
| 状态 | 效果 |
|------|------|
| 默认 | 背景渐变，轻微阴影 |
| 悬停 | 阴影加深，轻微放大 (scale 1.02) |
| 按下 | 阴影减弱，轻微缩小 (scale 0.98) |
| 禁用 | 灰色背景，无交互 |

### 输入框交互
- 聚焦时蓝色边框高亮 (`#0071e3`)
- 聚焦时轻微阴影
- 平滑过渡动画 (0.2s)

### 进度条动画
- 渐变背景流动效果
- 平滑百分比更新
- 完成时绿色填充动画

### 加载状态
- 旋转指示器
- 骨架屏占位
- 禁用相关按钮防止重复操作

### 完成反馈
- 成功提示动画（淡入淡出）
- 任务项绿色高亮
- 自动切换到"已完成"标签页

## 动画规范

### 过渡动画
```css
/* 通用过渡 */
transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1);

/* 弹性效果 */
transition: transform 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);
```

### 关键动画
- **淡入**: opacity 0→1, 0.3s
- **滑入**: transform translateY(10px)→0, 0.3s
- **缩放**: scale 0.95→1, 0.2s
- **进度条**: width 0→100%, 根据实际进度

## 组件详细设计

### 1. 主标题区
- 大标题 "PSiteDL" - 28px Bold
- 副标题 "视频下载工具" - 17px Regular, #86868b
- 左侧对齐，充足留白

### 2. URL 输入卡片
- 白色背景卡片，12px 圆角
- 大尺寸输入框，最小高度 80px
- Placeholder 文字 "#86868b"
- 支持多行 URL 输入

### 3. 设置卡片
- 网格布局，3 列
- 每个设置项独立卡片
- 下拉选择器采用苹果风格
- Checkbox 采用 iOS 风格开关

### 4. 操作按钮组
- 大尺寸按钮，最小高度 44px
- 主按钮（启动下载）：蓝色渐变背景
- 次按钮（加入待下载）：蓝色边框
- 危险按钮（清空）：红色文字
- 按钮间距 12px

### 5. 任务列表卡片
- 标签页切换设计
- 列表项悬停效果
- 进度条可视化
- 状态图标指示

### 6. 日志卡片
- 等宽字体显示日志
- 自动滚动到底部
- 支持清空操作
- 可折叠设计（可选）

## 实现技术要点

### tkinter 样式定制
- 使用 `ttk.Style()` 定制组件样式
- 自定义颜色配置
- 圆角通过图像实现或使用 `rounded_rectangle`

### 字体处理
- 优先使用系统字体
- macOS: SF Pro
- Windows: Segoe UI
- Linux: Cantarell / Ubuntu

### 高 DPI 支持
- 启用 DPI 感知
- 图标使用 SVG 或高分辨率 PNG
- 字体大小动态缩放
