# PSiteDL GUI 左右分栏布局设计文档

## 设计概述

将原有垂直布局重构为左右分栏布局，提升信息密度和操作效率。

---

## 布局结构

```
┌─────────────────────────────────────────────────────────────────┐
│  PSiteDL - 视频下载工具                              [窗口控制]  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────┐  ┌─────────────────────────────────┐  │
│  │                     │  │                                 │  │
│  │   左侧操作区 (60%)   │  │      右侧日志区 (40%)          │  │
│  │                     │  │                                 │  │
│  │  - URL 输入          │  │  - 全屏日志显示                 │  │
│  │  - 设置卡片          │  │  - 自动滚动                    │  │
│  │  - 任务列表          │  │  - 日志级别过滤                │  │
│  │  - 控制按钮          │  │  - 橙色文字 @ 深色背景          │  │
│  │                     │  │                                 │  │
│  └─────────────────────┘  └─────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 左侧操作区设计

### 1. 顶部 - URL 输入区
- **大尺寸文本框** (高度 120px)
- 支持多行输入
- 占位符提示
- 橙色高亮边框 (聚焦时)

### 2. 中部 - 设置卡片
采用卡片式布局，包含以下设置项:

| 设置项 | 类型 | 默认值 |
|--------|------|--------|
| 浏览器 | 下拉选择 | chrome |
| Profile | 文本输入 | Default |
| 输出目录 | 文本输入 + 浏览按钮 | ~/Downloads |
| 运行时探测秒数 | 数字输入 | 30 |
| 启用运行时探测 | 复选框 | ✓ |

### 3. 底部 - 任务管理区
使用标签页组织三种任务状态:

- **待下载**: 列表显示，支持批量操作
- **下载中**: 表格显示进度和状态
- **已完成**: 列表显示，支持快速访问

### 4. 控制按钮组
横向排列的主要操作按钮:

| 按钮 | 样式 | 功能 |
|------|------|------|
| 加入待下载 | Secondary | 将 URL 加入队列 |
| 启动下载 | Primary (橙色) | 开始处理队列 |
| 清空待下载 | Danger | 移除所有待处理任务 |
| 清空日志 | Secondary | 清除日志显示 |

### 5. 状态栏
- 实时显示下载状态
- 橙色高亮文字
- 显示：正在下载 | 待下载 | 已完成 数量

## 右侧日志区设计

### 视觉风格
- **背景色**: `#1e1e1e` (深色)
- **文字色**: `#ff9500` (橙色)
- **字体**: SF Mono 12px (等宽字体)
- **自动滚动**: 始终显示最新日志

### 日志级别
支持以下日志类型:

| 级别 | 前缀 | 颜色 |
|------|------|------|
| 信息 | `[info]` | 橙色 |
| 进度 | `[progress]` | 亮橙色 |
| 警告 | `[warning]` | 黄色 |
| 错误 | `[error]` | 红色 |
| 任务 | `[task-X]` | 青色 |

### 功能特性
- ✅ 自动滚动到最新日志
- ✅ 等宽字体对齐
- ✅ 深色背景减少眩光
- ✅ 橙色主题保持一致性

## 布局比例

### 默认比例
- **左侧操作区**: 60% (最小宽度 700px)
- **右侧日志区**: 40% (最小宽度 450px)

### 响应式设计
- 窗口宽度 < 1150px: 自动切换为垂直布局
- 窗口宽度 >= 1150px: 保持左右分栏

---

## 交互细节

### 圆角设计
- 按钮圆角：10px
- 卡片圆角：8px
- 输入框圆角：6px
- 进度条圆角：4px

### 分隔线
- 卡片之间：16px 间距
- 区域之间：24px 间距
- 分隔线颜色：`#353535`
- 分隔线高度：1px

### 悬停效果
- 按钮悬停：颜色变亮 15%
- 按钮按下：颜色变暗 10%
- 列表项悬停：背景高亮

## 技术实现

### 布局方式
使用 `tk.PanedWindow` 实现可调节的左右分栏:
- 支持拖动分割线调整比例
- 设置最小窗口尺寸限制
- 保持组件内边距一致性

### 组件复用
保留现有自定义组件:
- `DarkOrangeButton`
- `DarkOrangeCard`
- `DarkOrangeEntry`
- `DarkOrangeText`
- `DarkOrangeProgressbar`
- `DarkOrangeNotebook`

### 日志更新
- 队列异步更新
- 120ms 轮询间隔
- 自动滚动到末尾
