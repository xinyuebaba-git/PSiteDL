# PSiteDL 暗黑橙色风格设计文档

## 设计理念

采用**暗黑模式 + 橙色强调**的配色方案，打造专业、现代、统一的视觉体验。深色背景减少视觉疲劳，橙色主色调传递活力与科技感。

---

## 配色方案

### 背景色
| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#1a1a1a` | 深黑色，用于窗口背景 |
| 卡片背景 | `#2d2d2d` | 暗灰色，用于卡片/容器 |
| 输入框背景 | `#252525` | 略深于卡片，用于输入区域 |

### 主色调
| 用途 | 色值 | 说明 |
|------|------|------|
| 主橙色 | `#ff6b00` | 品牌主色，用于按钮/强调 |
| 次橙色 | `#ff8533` | 辅助橙色，用于渐变 |
| 亮橙色 | `#ffab40` | 高亮强调，用于 hover 状态 |

### 文字颜色
| 用途 | 色值 | 说明 |
|------|------|------|
| 主文字 | `#e0e0e0` | 浅灰色，用于正文 |
| 强调文字 | `#ff9500` | 橙色，用于标题/重点 |
| 次要文字 | `#a0a0a0` | 中灰色，用于副标题/提示 |
| 禁用文字 | `#505050` | 深灰色，用于禁用状态 |

### 边框与分隔
| 用途 | 色值 | 说明 |
|------|------|------|
| 边框 | `#404040` | 深灰色，用于卡片/输入框边框 |
| 分隔线 | `#353535` | 略深于边框，用于内容分隔 |

### 功能色
| 用途 | 色值 | 说明 |
|------|------|------|
| 成功 | `#4caf50` | 绿色，用于完成状态 |
| 警告 | `#ff9800` | 橙黄色，用于警告提示 |
| 错误 | `#f44336` | 红色，用于错误状态 |
| 进度条背景 | `#353535` | 深灰色，用于进度轨道 |

## 圆角设计规范

| 组件 | 圆角值 | 说明 |
|------|--------|------|
| 按钮 | `10px` | 统一圆角，友好点击区域 |
| 输入框 | `8px` | 略小圆角，保持专业感 |
| 卡片 | `12px` | 最大圆角，区分层级 |
| 边框/分隔 | `8px` | 与输入框一致 |
| 进度条 | `4px` | 较小圆角，精确感 |

## 组件设计规范

### 按钮 (DarkOrangeButton)
- **背景**: 橙色渐变 (`#ff6b00` → `#ff8533`)
- **文字**: 白色 (`#ffffff`)
- **Hover**: 亮橙色 (`#ffab40`)
- **Pressed**: 深橙色 (`#e65c00`)
- **Disabled**: 灰色 (`#404040`) + 禁用文字
- **高度**: 44px
- **最小宽度**: 120px

### 输入框 (DarkOrangeEntry)
- **背景**: `#252525`
- **边框**: `#404040` (1px)
- **聚焦边框**: `#ff6b00` (2px)
- **文字**: `#e0e0e0`
- **占位符**: `#707070`
- **插入光标**: `#ff6b00`

### 卡片 (DarkOrangeCard)
- **背景**: `#2d2d2d`
- **边框**: `#404040` (1px)
- **内边距**: 20px
- **圆角**: 12px
- **标题**: 橙色 (`#ff9500`), 14pt 粗体

### 进度条 (DarkOrangeProgressbar)
- **轨道**: `#353535`
- **进度**: 橙色渐变 (`#ff6b00` → `#ffab40`)
- **高度**: 8px
- **圆角**: 4px

### 文本框 (DarkOrangeText)
- **背景**: `#252525`
- **边框**: `#404040`
- **文字**: `#e0e0e0`
- **内边距**: 12px

### 标签页 (DarkOrangeNotebook)
- **背景**: `#2d2d2d`
- **选中标签**: 橙色背景 + 白色文字
- **未选中标签**: 深灰背景 + 浅灰文字
- **圆角**: 10px (顶部)

### 列表框 (DarkOrangeListbox)
- **背景**: `#252525`
- **选中背景**: `#ff6b00`
- **选中文字**: 白色
- **边框**: 无

## 交互反馈

### Hover 状态
- 按钮：亮度提升 10%
- 输入框：边框高亮橙色
- 列表项：背景加深

### Pressed 状态
- 按钮：亮度降低 15%
- 轻微缩放效果 (可选)

### Disabled 状态
- 按钮：灰色背景 + 禁用文字
- 输入框：深灰背景 + 禁用文字
- 无 Hover/Click 反馈

### 进度反馈
- 进度条：橙色渐变填充
- 状态文字：橙色高亮
- 完成提示：绿色成功状态

## 可访问性

### 对比度要求
- 正文文字与背景：≥ 4.5:1
- 大标题与背景：≥ 3:1
- 橙色按钮与白色文字：≥ 4.5:1

### 键盘导航
- Tab 键顺序：从上到下，从左到右
- Enter 键：触发默认按钮
- Escape 键：关闭对话框

### 状态提示
- 所有操作提供明确反馈
- 错误信息清晰可见
- 进度状态实时更新

## 设计原则总结

1. **统一性**: 所有组件遵循同一配色和圆角规范
2. **对比度**: 确保文字清晰可读，符合无障碍标准
3. **一致性**: 交互反馈统一，用户预期明确
4. **现代感**: 渐变、圆角、暗色主题符合当代设计趋势
5. **品牌感**: 橙色作为品牌识别色，贯穿整个界面
